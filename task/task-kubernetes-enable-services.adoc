:allow-uri-read: 
:nofooter:

[role="lead"]
管理対象のKubernetesクラスタをCanvasに追加すると、ネットアップのクラウドデータサービスを使用して高度なデータ管理を行うことができます。

link:https://docs.netapp.com/us-en/bluexp-backup-recovery/concept-kubernetes-backup-to-cloud.html["BlueXPのバックアップとリカバリを使用してKubernetesクラスタのデータを保護する方法をご紹介します"^]。
